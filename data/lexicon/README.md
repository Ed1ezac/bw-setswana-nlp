# Lexicon Dataset

Structured Setswana (`tn-BW`) dictionary dataset: words, definitions, part-of-speech tags, noun class annotations, and example sentences.

## Files

| File | Description |
|------|-------------|
| `lexicon.json` | The lexicon dataset — an array of entry objects |
| `schema.json` | JSON Schema (draft-07) defining the structure of each entry |
